## 🎮 寶可夢卡牌對戰

### 遊戲方式
1. 點擊「開始遊戲」，系統隨機分配雙方各 3 隻寶可夢
2. 出戰前可選擇換掉其中一隻（限換一次）
3. 輪流使用技能攻擊，消滅對手全部寶可夢即獲勝

### 特色
- **20 隻寶可夢**，涵蓋火、水、草、電、超能、格鬥、幽靈、龍、鋼、冰、一般、惡、地面、飛行共 14 種屬性
- **屬性相剋系統**：攻擊效果依屬性有 ×2 / ×0.5 / ×0 三種變化
- **訓練師手牌**：每場可使用道具牌（傷藥、攻擊強化、防禦強化、能量強化、復活藥）與支援者牌（治療師、換人命令、全力出擊）
- **Buff 視覺提示**：使用強化牌後，場上即時顯示當前加成
- **備戰欄換場**：點擊備戰寶可夢可主動換場（消耗一回合）
- **電腦 AI**：自動判斷補血、強化、換場時機

### 收錄寶可夢
固拉多、蓋歐卡、列空座、密勒頓、故勒頓、鳳王、路奇亞、噴火龍、水箭龜、妙蛙花、超夢、耿鬼、怪力、快龍、胡地、卡比獸、風速狗、拉普拉斯、暴鯉龍、雷丘

## 🚀 開啟方式

**直接開啟：**
```
# macOS
open pokemon_battle.html

# Windows
start pokemon_battle.html
```

**或雙擊捷徑檔：**
- macOS → `start.command`
- Windows → `start.bat`
